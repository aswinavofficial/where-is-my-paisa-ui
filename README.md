# Where is my Paisa - Landing Page

A modern, responsive landing page for the **Where is my Paisa** Android app — a personal finance tracker that helps users manage expenses, set budgets, and gain financial insights.

## 🚀 Tech Stack

- **Next.js 16** — React framework for production
- **TypeScript** — Type-safe development
- **Tailwind CSS 4** — Utility-first styling
- **Framer Motion** — Smooth animations
- **Lucide React** — Beautiful icons

## 🏃 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## 📱 Sections

- **Hero** — App introduction with phone mockup and download CTA
- **Features** — Key app capabilities (expense tracking, analytics, budgets, etc.)
- **Screenshots** — Interactive app preview with phone mockups
- **How It Works** — 4-step guide to getting started
- **Testimonials** — User reviews and ratings
- **Download CTA** — Final call-to-action with Play Store and GitHub links
- **Footer** — Navigation, resources, and social links
